ما هي واجهات API هذه؟

هذه هي الواجهات الخلفية التي تجعل Study Hub منصة ذكية. يمكن لأي مطور استخدام هذه الواجهات لبناء تطبيقات إضافية (مثل تطبيق موبايل) تتكامل مع Study Hub. جميع الواجهات مؤمنة وتتطلب توكن المصادقة.

مميزات WebRTC و WebSockets

إلى جانب واجهات REST API المعروضة، تدعم المنصة اتصالات لحظية باستخدام WebSockets (للدردشة والإشعارات) و WebRTC (لمكالمات الفيديو المباشرة). هذه التقنيات تتيح تجربة تعاونية فورية دون الحاجة لتحديث الصفحة.
